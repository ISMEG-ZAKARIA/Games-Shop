# Games Shop

A professional game store website with a modern UI and a fully functional shopping cart. This project is a complete front-end implementation for an online game store, featuring game listings, product categories, and a multi-step checkout process.

## Features

- **Dynamic Shopping Cart**: Add games to your cart and manage them with ease.
- **Product Categories**: Easily browse games by genre and platform.
- **Multi-Step Checkout**: A streamlined process for completing orders with form validation.
- **Responsive Design**: Optimized for all devices and screen sizes.
- **Form Validation**: Real-time validation for personal info, address, and payment details.
- **Privacy & Terms**: Dedicated pages for legal compliance.

## Tech Stack

- **HTML5**: Structured content for a rich user experience.
- **CSS3**: Custom styles for a modern and engaging interface.
- **JavaScript (Vanilla)**: Core logic for cart management and checkout.
- **Fonts**: Roboto from Google Fonts for clean typography.

## Installation

1.  **Clone the Repository**:
    ```bash
    git clone https://github.com/Games-Shop/Games-Shop.git
    cd Games-Shop
    ```
2.  **Open in Browser**:
    Open `index.html` directly in your browser or use a live server for the best experience.

## Environment Variables Guide

Since this is a front-end project, no sensitive information is stored in environment variables. However, you can use `.env.example` for any future API integrations.

## How to Run Locally

You can run this project locally using any static file server:

-   **Live Server (VS Code Extension)**: Recommended for the best development experience.
-   **NPM Serve**:
    ```bash
    npm install -g serve
    serve .
    ```

## How to Deploy on Vercel

1.  **Push to GitHub**: Connect your GitHub account to Vercel.
2.  **Import Project**: Select the `Games-Shop` repository.
3.  **Deploy**: Vercel will automatically detect the static project and deploy it.
